# Consumer-Electronics-Sales-Analysis
EDA and ML project analyzing consumer electronics sales data using python

## Overview
Analyzed a consumer electronics dataset to understand customer behavior and purchase intent.Project includes EDA,visualization,and a baseline ML model.

## Dataset
Contains 9,000  rows with features like:
- ProductCategory, ProductBrand, ProductPrice  
- CustomerAge, CustomerGender, PurchaseFrequency  
- CustomerSatisfaction, PurchaseIntent (target)
  
**Source:** Dataset available on Kaggle: [Consumer Electronics Sales Data]-https://www.kaggle.com/datasets/rabieelkharoua/consumer-electronics-sales-dataset)  

## What I Did
- Performed **EDA** to explore patterns and clean data  
- Created visualizations (bar charts, boxplots, histograms, heatmaps)  
- Built a **Logistic Regression** model to predict purchase intent  
- Evaluated performance using **accuracy and confusion matrix**

## Key Insights
- Higher satisfaction → higher purchase intent  
- Frequent buyers are more likely to purchase  
- Product category and price influence buying decisions  

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, **Google Colab**
